# SolarFocuser (IN PROGESS)
This repo is for a simulator inspired by the work of McIness et al. 2017 (https://eprints.gla.ac.uk/135798/) in which a robotic solar sail is controlled using MPC (for now)
Controller is inspired from the Computational Control class at ETH Zurich by Saverio Bolognani.

# Installation Instructions

git clone repo
All required packages are in pyproject.toml
Set up a conda environment and run pip install -e .
